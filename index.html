Introduction

Hello! I'm Sze Ting, an aspiring game developer. This page features some of my most prominent works.


Prop Hunt

In this project, I implemented a simple prop hunt game featuring multiplayer and server programming.

This was done with Unity3D and Photon (PUN 2). The video shows the gameplay and perspectives of both players.


Asteroid Shooter

This is a simple 2D asteroid shooter game featuring realistic physics.

Done with OpenGL and C++ entirely, all the physics implementations, including inelastic collisions, acceleration and gravity, are done through writing the formulae into code.

It also has a widespread variety of mechanics, such as powerups and a scoring system.


Mobyland

Designed and programmed by my team of 4, Mobyland is a fast-paced survival game featuring weapons and enemies.

Having done the camera, player, and combat mechanics, I put a lot of effort in perfecting my parts and the project turned out successful despite many setbacks.
